## Setup/Installation Requirements

##### Create Database and Tables
* Clone from GitHub
* In a command window:
```
> $ cd {folder name}
> $ npm init
> $ npm install gulp --save-dev
> $ npm install
> $ bower init
> $ bower install
> $ gulp build
> $ gulp serve
```
* Requires Node.js
* Navigate to http://localhost:3000 in web browser of choice
